# 内容生产日程表

## 每日流程

| 时间 | 事项 | 工具 | 说明 |
|------|------|------|------|
| 08:00 | 大脑训练 | 飞书推送 | 舒尔特+斯特鲁普，保持专注力 |
| 09:00 | 浏览热点/灵感收集 | 抖音/小红书 | 刷同赛道内容，记录爆款选题灵感 |
| 10:00 | Agent1出选题 | 元器 | 输入关键词→8个选题→人工筛选1-2个 |
| 10:30 | 热度验证 | 抖音搜索 | 搜选题关键词，确认有流量且未泛滥 |
| 11:00 | Agent2写文案 | 元器 | 输入选题+框架→2-5分钟文案→人工审改 |
| 12:00 | 美股日报（交易日） | 飞书推送 | Agent自动推送 |
| 14:00 | Agent3出标题 | 元器 | 输入文案→8-10个标题→选TOP1-2 |
| 14:30 | 拍摄/录制 | 手机 | 按文案口播，注意节奏和情绪 |
| 15:30 | 剪辑发布 | 剪映 | 封面用爆款对标，标题用Agent3输出 |
| 16:00 | 数据复盘 | 抖音后台 | 记录播放量、完播率、互动数据 |
| 00:00 | 每日复盘 | 飞书推送 | Agent推送复盘模板→填写→保存 |

## 每周流程

| 日期 | 事项 | 说明 |
|------|------|------|
| 周一 | 本周选题规划 | 用Agent1批量出10-20个选题，筛选本周5-7个 |
| 周三 | 中期数据复盘 | 看本周已发内容数据，调整选题方向 |
| 周五 | 批量录制日 | 集中录制2-3条，减少每天拍摄压力 |
| 周日 | 周复盘+下周规划 | 总结本周爆款/扑街原因，更新选题库 |

## Agent使用流程

```
关键词 → Agent1选题 → 人工筛选 → Agent2文案 → 人工审改 → Agent3标题 → 人工选定 → 拍摄 → 发布
```

## 关键指标追踪

### 内容指标
- 2秒跳出率：目标越低越好（<40%为佳）
- 整体完播率：目标 >30%
- 点赞率：目标 >3%
- 评论率：目标 >0.5%

### 生产效率
- 日产出：目标1条/天
- 从选题到发布：目标 <4小时
- Agent文案一次通过率：目标 >60%
