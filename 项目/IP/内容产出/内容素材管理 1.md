# 内容素材管理

## 一、知识库素材

### 1. 后方知识储备
- 来源：书本上传
- 形式：完整结构化课程

### 2. 前方数据储备
- 爆款选题
- 爆款内容框架
- 爆款标题

### 3. 后端产品内容储备
- 避免实时调用大模型 token 消耗
- 知识库内容与后端产品内容高度吻合
- 可实现共享

---

## 二、Agent团队分配

### Agent 1：选题
- 功能：对应生产流程一
- 负责：观点生成

### Agent 2：内容输出
- 输入：前方数据储备2（爆款框架）
- 底层：RAG框架
- 负责：精准实现文案输出

### Agent 3：标题设计
- 输入：Agent 2 输出内容
- 底层：前方数据储备3（产品内容）
- RAG框架
- 负责：精准输出标题

### Agent 4：产品内容管理
- 负责：后端产品内容储备维护

---

## RAG框架设计要点

### 核心原则
- 限制AI发散思维
- 避免通用语言模型胡编乱造
- 输出词不达意 → 提示词越精确越好

### 提示词设计要求
- 需根据各agent工作内容设计
- 限制AI发散思维
- 避免通用语言模型胡编乱造
- 输出词不达意
- 提示词越精确越好
